# Clean-up the Guestroom – Robotics AI Task

This repository defines a Robotics AI task
for the Konnex simulator.

## Task
Clean-up the guestroom.

## Objective
Evaluate robotic planning, object recognition,
and environmental interaction in a guestroom scenario.

## Scenario
The robot receives a request to clean and organize
a guestroom within a simulated environment.

## Key Actions
- Identify misplaced objects
- Organize furniture alignment
- Collect trash items
- Prepare room for guest readiness

## Environment
Konnex AI Miner Testnet
